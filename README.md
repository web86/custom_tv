# custom_tv
Вывод в админке кастомных TV

В файле common_steps.txt описаны шаги для создания кастомного TV

сделал 2 варианта кастомного вывода TV который отображает указанные родители как категории соблюдая всю иерархию родителей, оба варианта имеют поиск по дереву, подсветку найденного ресурса, создание чипсов по клику и по enter, ограничение по максимальному количесвту выбранных элементов

1 вариант modal - покажется в красовм модальном окне
2 вариант dropdown - покажет в выпадающем поле
